Numpy
=====
Numpy是一个关于科学计算的十分重要的第三方库，许多库以此为基础。包括：
- 高效的N维数组
- 复杂的函数
- 集成C/C++工具
- 高效线性代数，傅里叶变化，随机数计算能力
- 高效的多维自定义数据类型
